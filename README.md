# AI-Powered Resume Builder (Frontend)

## Overview
The **AI-Powered Resume Builder** is a web application that allows users to create professional, ATS-friendly resumes in real-time. The frontend is developed using modern web technologies and deployed on AWS S3 with CI/CD automation via CodePipeline.

This project provides an intuitive interface for users to input their information, preview their resume in real-time, and download it in a professional format.

---

## Technologies Used
- **Frontend:** HTML, CSS, JavaScript  
- **Cloud Hosting:** AWS S3  
- **CI/CD:** AWS CodePipeline  
- **Version Control:** GitHub  

---

## Features
- **Responsive Forms:** Easy-to-use input forms for personal details, education, experience, and skills.  
- **Real-Time Preview:** Instant preview of the resume as you fill in details.  
- **ATS-Friendly Templates:** Templates optimized for applicant tracking systems.  
- **Cloud Deployment:** Hosted on AWS S3 for reliable and scalable access.  
- **CI/CD Integration:** Automated deployment using AWS CodePipeline linked with GitHub.
